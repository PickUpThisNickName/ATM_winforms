Программа «Банкомат» предназначена для эмуляции работы с банкоматом.

Среди доступных купюр присутствуют купюры достоинства 50, 100, 200, 500, 1000, 2000, 5000р
(Наиболее часто используемые купюры в обороте на момент разработки программы - 2022г.)

Существует ограничение по количеству вносимых купюр, которое составляет 100 купюр. После этого
банкомат новые купюры принимать не сможет.

Стартовые значения купюр каждого достоинства в банкомате - 10шт.
Стартовое значение баланса пользователя банкомата - 1500р.

Стартовые значения инициализируются и хранятся в памяти только во время сессии работы с банкоматом.

Приложение разработано на платформе .net framework с использованием технологии winforms.
Для запуска программы необходим пакет .net runtime 6.0, для компиляции проекта необходим пакет .net sdk 6.0.

Исходный код проекта включает IDE для работы с банкоматом, реализованное классом ATM. Номинальные значения купюр
а также их количество при старте можно переопределить при создании объекта класса в MainForm.cs
